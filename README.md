# Banking App Website

## Overview
The Banking App Website is a static web application designed to provide users with a seamless experience for banking transactions. The application includes a login page, a dashboard with tabs for Direct Branch Transfer and Deposit/Withdraw details, and dedicated pages for deposit and withdrawal processes.

## Features
- **User Authentication**: Secure login functionality to access the dashboard.
- **Dashboard Navigation**: Two tabs for easy navigation between Direct Branch Transfer and Deposit/Withdraw details.
- **Transaction Processes**: Dedicated pages for deposit and withdrawal processes, including forms for user input.
- **Cheque Management**: A page for managing cheque details with options to reset and save.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to access the application.
3. Navigate through the application using the provided links and buttons.
